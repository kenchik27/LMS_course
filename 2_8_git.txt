git status выводит текущее состояние репозитория

Добавление
git add сообщаем git-у, что планируем зафиксировать определенные (или все) файлы. Добавим все файлы в папке:

git add .

git commit - фиксирует текущие изменения
Сопровождается комментарием

Когда мы делаем commit, git запоминает текущее состояние файла:

git commit -m "Добавил файл для примера работы с git"
